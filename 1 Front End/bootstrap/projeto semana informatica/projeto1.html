Sobre o Projeto

Este projeto foi desenvolvido durante a Primeira Semana de Informática. O objetivo do projeto é demonstrar as habilidades dos alunos em desenvolvimento web usando Bootstrap.

Fotos do Projeto

Descrição do Projeto

O projeto consiste na criação de um site utilizando Bootstrap. Os alunos aprenderam a usar componentes como cards, navegação, modals, e muito mais. A aplicação é responsiva e segue boas práticas de desenvolvimento web.

Alunos Envolvidos

  • Aluno 1 - Desenvolvimento Front-end
  • Aluno 2 - Design e UX
  • Aluno 3 - Desenvolvimento Back-end
Sobre o Evento

A Primeira Semana de Informática é um evento voltado para a disseminação de conhecimentos em tecnologia e inovação.
